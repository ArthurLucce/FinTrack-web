# FinTrack — Dashboard

Projeto desenvolvido para a disciplina de ADS na FIAP.

## Como rodar

1. Faça o download ou clone o repositório
2. Abra o arquivo `index.html` diretamente no navegador (não precisa de servidor)
3. Nenhuma dependência local — o Tailwind CSS é carregado via CDN

## Estrutura de arquivos

```
fintrack-web/
├── index.html   → estrutura HTML da tela Dashboard
├── style.css    → estilos customizados separados do HTML
└── README.md    → este arquivo
```

## Tecnologias

- HTML5 semântico
- CSS3 (arquivo separado — style.css)
- Tailwind CSS via CDN
- Responsivo: desktop, tablet e mobile

## Comandos Git utilizados para subir o projeto

```bash
git init
git add .
git commit -m "feat: dashboard FinTrack com Tailwind CSS"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/fintrack-web.git
git push -u origin main
```
